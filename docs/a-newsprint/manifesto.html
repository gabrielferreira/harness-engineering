AI coding assistants entregam o que se pede. O problema raramente é o modelo. É a ausência de infraestrutura que diga onde ele está, o que o time valoriza, e como o trabalho é avaliado.

AI coding assistants deliver what you ask. The problem is rarely the model. It is the absence of infrastructure that tells it where it stands, what the team values, and how the work is judged.

Harness é essa infraestrutura. Este documento declara como pensamos sobre ela. Harness is that infrastructure. This document declares how we think about it.

§ I — O que valorizamos§ I — What we value★

O que valorizamos.

What we value.

Reconhecemos valor em ambos os lados. Quando há tensão, preferimos o primeiro. We see value on both sides. When tension arises, we prefer the first.
