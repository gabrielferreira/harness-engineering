Welcome

Harness Engineering é um destilado de práticas — não uma metodologia fechada nem um produto. É um repositório aberto que reúne o que vem funcionando para times que usam AI coding assistants em código de produção.

Harness Engineering is a distillation of practices — not a closed methodology nor a product. It is an open repository gathering what has been working for teams that use AI coding assistants in production code.

A premissa é simples: o problema raramente é o modelo. É a infraestrutura ao redor dele — contexto, regras, especialização, verificação, continuidade — que define se o AI ajuda ou atrapalha.

The premise is simple: the problem is rarely the model. It is the infrastructure around it — context, rules, specialization, verification, continuity — that decides whether AI helps or harms.

Os princípios são agnósticos à ferramenta: aplicam-se a Claude Code, Cursor, GitHub Copilot, Continue, Aider, Windsurf, Cline — ou a qualquer combinação.

The principles are tool-agnostic: they apply to Claude Code, Cursor, GitHub Copilot, Continue, Aider, Windsurf, Cline — or any combination.
